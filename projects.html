GEOG5995M Programming for Social Science: Core Skills

Building up an Agent-Based Model (ABM)


During the course's practicals of this module a simple but generic Agent-based model (ABM) was built up in Python (programming language). It simulates the actions and interactions of autonomous agents with the environment and with each other.

In building this ABM, broad coding concepts of use across science were covered such as structuring code, utilising rules and maths, getting code to the interact with other code, interacting with users, and reading and writing data.

To picture it as a real-life process, the agents can be imagined as sheep wandering around an environment (meadow), nibbling away data (grass). In the model, the sheep interact with the environment (feeding from it) and with each other (checking that there are no others in their environment).

Essentially, the building up of the model can be divided into four different steps:

  • Step 1 - The creation of the agents

    • In practicals 1 to 5 the agents were built. We created them, made them move, and gave them information about other agents, as well as attributes and behaviour. In parallel with these process we learned some code shrinking technics and the concepts of functions, classes and objects. In the graph on the left we can see the stage of the model at the end of the practical 5.





  • Step 2 - The environment

    • In practical 6 the environment of the model was created importing data from a csv file. In addition, we made the agents to have access and interact with it - nibbling away data and building up stores. On the left the environment plotted in this practical can be seen.







  • Step 3 - Communication

    • Practical 7 was focused on the communication between agents. To do this agents needed to know about each other, to check the environment and to scan the list of other agents to know if there were others in their neighbourhood. In the graph on the left it the gradual erosion of data through the random movement of agents in the environment can be seen.





  • Step 4 - Animation and other enhancements

    • And finally, in practical 8 the model was animated and some ideas were given in order to enhance it.
