GEOG5995M Programming for Social Science: Core Skills

Building up an Agent-Based Model (ABM)

During the course's practicals of this module a simple but complete Agent-based model (ABM) was built up in Python (programming language).

In building this ABM, broad coding concepts of use across science such as structuring code, utilising rules and maths, getting code to the interact with other code, interacting with users, and reading and writing data were covered.

To picture the model as a real-life process, the agents can be imagined as sheep wandering around an environment (meadow) and nibbling away data (grass). In this way, the sheep interact with the environment (feeding from it) and with each other (checking if there are others around).

Essentially, the building up of the model can be divided into four different steps:

Step 1 - The creation of the agents

In practicals 1 to 5 we created the agents, made them move, gave them information about others, as well as attributes and behaviour. In parallel with this process, we learned some code shrinking technics and the concepts of functions, classes and objects. In the graph on the left, we can see the stage of the model at the end of the practical 5.





Step 2 - The environment

In practical 6, the environment of the model was created importing data from a csv file. In addition, we made the agents to have access and interact with it - nibbling away data and building up stores. On the left, the environment is plotted.







Step 3 - Communication

Practical 7 was focused on the communication between agents. To do this, agents needed to check the environment, to scan the list of other agents, and to know if there were others in their neighbourhood. The graph on the left shows the gradual erosion of data through the random movement of agents in the environment.






Step 4 - Animation and other enhancements

And finally, practical class 8 was to animate the code in order to help on the users' visualisation, as well as to enhance the model with extra work.
