# eugenividal.github.io
Eugeni's website

This is the repository for an online portfolio of the practical work for the module GEOG5995M Programming for Social Science: Core Skills. 
The online portfolio comprises a website of four pages. In pages one (home) and two (research) you will find biographical material explaining who I am and what my PhD research project is about. In page three (projects) details of the Agent-Based Model built in the practicals classes of the module as well as a link to the code. Finally, in page four (contact) there are the links to my email, and GitHub and LinkedIn accounts.
